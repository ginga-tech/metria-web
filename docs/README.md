# Metria Web - Documentation Index

This folder is the single source of truth for frontend documentation.

## Naming Convention

- Stable docs: `NN-Topic.md` (numeric order by domain/context priority)
- Backlog items: `99-Backlog.md`
- Entry point: `README.md`

## Documents

- `00-Overview.md`: product purpose, scope, stack
- `01-Architecture.md`: folder organization and runtime flow
- `02-Routing-and-Screens.md`: routes and layout
- `03-Auth-Flow.md`: auth and OAuth flow
- `04-Billing-and-Checkout.md`: Stripe checkout/subscription flow
- `05-Goals-and-SubGoals.md`: goals/sub-goals behavior and rules
- `06-Environment-Variables.md`: required and optional env vars
- `07-API-Contracts.md`: frontend-consumed endpoints and payloads
- `08-Testing-Strategy.md`: smoke and integration checks
- `09-Local-Execution.md`: local run/build commands
- `10-Deployment-Railway.md`: deploy and DNS checklist
- `11-ADR-Decisions.md`: architectural decisions and trade-offs
- `12-Development-Standards.md`: coding and collaboration standards
- `13-AI-Guidelines.md`: practical guidelines for AI-assisted changes
- `99-Backlog.md`: pending ideas and improvements

## Quick Start

1. Read `00-Overview.md`
2. Read `01-Architecture.md`
3. Configure env with `06-Environment-Variables.md`
4. Execute local setup from `09-Local-Execution.md`
5. For billing work, read `04-Billing-and-Checkout.md`
